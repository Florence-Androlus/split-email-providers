Split Email Providers is a WooCommerce plugin that sends automated emails to suppliers for each order, streamlining communication.

== Description ==

🌟 **Split Email Providers** is an innovative WooCommerce plugin designed to streamline order tracking for store owners working with multiple suppliers. This plugin sends automated emails for each order to the respective suppliers, eliminating the need for a marketplace and making the process simpler, faster, and more efficient. Perfect for businesses that want to maintain direct relationships with their suppliers while using WooCommerce.

Key Features:
* 📬 **Automated email notifications to suppliers**: For each order, emails are sent to the relevant suppliers with the details of their products.
* 🔄 **Seamless integration**: Works directly with WooCommerce without requiring complex configuration.

== Installation ==

1. Download the plugin.
2. 📂 Go to your WordPress dashboard > Plugins > Add New and upload the plugin file.
3. 📲 Activate the plugin on the "Plugins" page to start using it.

== Frequently Asked Questions ==

= 🛠️ Is this plugin only for WooCommerce? =
Yes, Split Email Providers is specifically designed for WooCommerce stores.

= 📝 Can the content of the emails sent to suppliers be customized? =
No, the plugin currently does not include options to customize the email message or details.
