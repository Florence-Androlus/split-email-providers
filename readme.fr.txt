Split Email Providers est un plugin WooCommerce qui envoie des emails automatisés aux fournisseurs pour chaque commande, simplifiant ainsi la communication.

== Description ==

🌟 **Split Email Providers** est un plugin WooCommerce innovant qui optimise le suivi des commandes pour les gérants de boutiques ayant plusieurs fournisseurs. Ce plugin envoie des emails automatisés pour chaque commande, à chaque fournisseur correspondant. Plus besoin de passer par une marketplace, ce qui rend le processus plus simple, rapide et efficace. Idéal pour les entreprises qui souhaitent garder une relation directe avec leurs fournisseurs tout en utilisant WooCommerce.

Fonctionnalités principales :
* 📬 **Envoi d’emails automatiques aux fournisseurs** : À chaque commande, les emails sont envoyés aux fournisseurs concernés avec les détails de leurs produits.
* 🔄 **Intégration fluide** : Fonctionne directement avec WooCommerce sans nécessiter de configuration complexe.

== Installation ==

1. Téléchargez le plugin.
2. 📂 Rendez-vous dans votre tableau de bord WordPress > Extensions > Ajouter et téléversez le fichier.
3. 📲 Activez le plugin dans la page "Extensions" pour commencer à l’utiliser.

== Frequently Asked Questions ==

= 🛠️ Est-ce que ce plugin est uniquement pour WooCommerce ? =
Oui, Split Email Providers est conçu spécifiquement pour les boutiques WooCommerce.

= 📝 Peut-on personnaliser le contenu des emails envoyés aux fournisseurs ? =
Non, le plugin n'inclut pas pour l'instant d'options pour personnaliser le message et les détails de l'email.
